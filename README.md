# Edulearn-LMS-Project
Documentation and system architecture for EduLearn, a collaborative Learning Management System (LMS) project.
## My Participation
As part of the project team, I contributed to:
Theoretical System Design: Mapping out the logical framework and architectural requirements for the LMS.
Technical Analysis: Conducting research on system functionalities and user requirements.
Group Brainstorming: Participating in defining the core features and data flow.
Documentation: Assisting in the preparation of the final project report and theoretical specifications.
